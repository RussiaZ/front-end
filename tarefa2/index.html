Métodos de Ordenação em C

Os métodos de ordenação (ou algoritmos de ordenação) são procedimentos computacionais que reorganizam um conjunto de elementos segundo uma ordem definida — crescente, decrescente ou lexicográfica. Em C, esses algoritmos são aplicados diretamente sobre vetores (arrays) e são fundamentais para a eficiência de sistemas de busca, bancos de dados e processamento de dados.

Os métodos de ordenação são separados em dois grupos, os métodos simples (Bubble, Selection e Insertion), e os métodos complexos (Quick e Merge), os métodos mais simples, ao contrário dos complexos, são mais fáceis de implementar no código, mas, tem eficiência reduzida comparado aos métodos mais complexos.

Bubble Sort

Um dos algoritmos mais intuitivos. Ele percorre repetidamente o vetor, comparando pares de elementos adjacentes e trocando-os quando estão fora de ordem. A cada passagem completa, o maior elemento restante "sobe" para sua posição final — daí o nome bolha.

// funcionamento passo a passo

  1. Compare o elemento da posição i com o da posição i+1.
  2. Se estiverem fora de ordem, troque-os.
  3. Repita para todos os pares adjacentes do início ao fim.
  4. A cada passagem, o vetor ativo é reduzido em 1 (o último elemento já está ordenado).

// implementação em C

void bubbleSort(int v[], int n) {
    int i, j, temp;
    for (i = 0; i < n - 1; i++) {
        for (j = 0; j < n - 1 - i; j++) {
            if (v[j] > v[j + 1]) {
                temp    = v[j];
                v[j]    = v[j + 1];
                v[j+1]  = temp;
            }
        }
    }
}

Selection Sort

O Selection Sort divide o vetor em duas partes: a parte ordenada (à esquerda) e a parte desordenada (à direita). A cada iteração, ele encontra o menor elemento da parte desordenada e o coloca no final da parte ordenada.

// implementação em C

void selectionSort(int v[], int n) {
    int i, j, minIdx, temp;
    for (i = 0; i < n - 1; i++) {
        minIdx = i;
        for (j = i + 1; j < n; j++)
            if (v[j] < v[minIdx]) minIdx = j;
        temp      = v[minIdx];
        v[minIdx] = v[i];
        v[i]      = temp;
    }
}

Insertion Sort

O Insertion Sort constrói o vetor ordenado elemento por elemento. A cada passo, o elemento atual é inserido na posição correta dentro da parte já ordenada. É análogo a como as pessoas ordenam cartas de baralho na mão.

// implementação em C

void insertionSort(int v[], int n) {
    int i, key, j;
    for (i = 1; i < n; i++) {
        key = v[i];
        j   = i - 1;
        while (j >= 0 && v[j] > key) {
            v[j + 1] = v[j];
            j--;
        }
        v[j + 1] = key;
    }
}

Quick Sort

Proposto por Tony Hoare em 1959, o Quick Sort é o algoritmo de ordenação de propósito geral mais amplamente utilizado. Ele adota a estratégia dividir para conquistar: escolhe um elemento como pivô e particiona o vetor em dois sub-vetores — elementos menores à esquerda e maiores à direita — depois ordena recursivamente cada parte.

// implementação em C

int partition(int v[], int low, int high) {
    int pivot = v[high], i = low - 1, j, temp;
    for (j = low; j < high; j++) {
        if (v[j] <= pivot) {
            i++;
            temp = v[i]; v[i] = v[j]; v[j] = temp;
        }
    }
    temp = v[i+1]; v[i+1] = v[high]; v[high] = temp;
    return i + 1;
}

void quickSort(int v[], int low, int high) {
    if (low < high) {
        int pi = partition(v, low, high);
        quickSort(v, low, pi - 1);
        quickSort(v, pi + 1, high);
    }
}

Merge Sort

Criado por John von Neumann em 1945, o Merge Sort também usa dividir para conquistar. Ele divide o vetor ao meio recursivamente até ter sub-vetores de 1 elemento, depois os intercala (merge) em ordem. É garantidamente O(n log n) em todos os casos, mas exige memória auxiliar O(n).

// implementação em C

void merge(int v[], int l, int m, int r) {
    int n1 = m-l+1, n2 = r-m, i, j, k;
    int L[n1], R[n2];
    for (i=0; i<n1; i++) L[i] = v[l+i];
    for (j=0; j<n2; j++) R[j] = v[m+1+j];
    i = j = 0; k = l;
    while (i < n1 && j < n2)
        v[k++] = (L[i] <= R[j]) ? L[i++] : R[j++];
    while (i < n1) v[k++] = L[i++];
    while (j < n2) v[k++] = R[j++];
}

void mergeSort(int v[], int l, int r) {
    if (l < r) {
        int m = l + (r - l) / 2;
        mergeSort(v, l, m);
        mergeSort(v, m+1, r);
        merge(v, l, m, r);
    }
}
