🎯 Visão Geral

Sistema de telemetria em tempo real para kartismo. Coleta dados de sensores no kart (RPM, velocidade GPS, temperatura, umidade) via ESP32-C3, transmite sem fio por ESP-NOW para um display receptor de 3.5" (CYD ESP32), com dashboard visual ao vivo. Futuramente integra Raspberry Pi para logging e dashboard web.

📦 Protocolo de Comunicação

Estrutura do Pacote Telemetry (ESP-NOW)

Packed struct · Canal 6 · Peer-to-peer sem roteador · Frequência: ~20 Hz (a definir)

# Campo Tipo Tamanho Descrição Unidade
1 ms uint32_t 4 bytes Timestamp desde boot (millis) ms
2 rpm float 4 bytes Rotações por minuto do motor RPM
3 speed float 4 bytes Velocidade via GPS km/h
4 lat double 8 bytes Latitude GPS graus
5 lon double 8 bytes Longitude GPS graus
6 sats uint8_t 1 byte Número de satélites GPS —
7 temp float 4 bytes Temperatura ambiente DHT11 °C
8 hum float 4 bytes Umidade relativa DHT11 %
Total 37 bytes Muito abaixo do limite ESP-NOW (250 bytes). Espaço para expansão.
